Student Database Application

This application should do the following:
* Ask the user how many new students will be added to the database.
* The user would be prompted to enter the name and year for each student.
* The student would have a 5-digit unique ID, with the first number being their grade level.
* Each course costs $600 to enroll.
* The student able to view their balance and pay the tuition.
* You can see the Student status (name, ID, courses enrolled, and balance).
